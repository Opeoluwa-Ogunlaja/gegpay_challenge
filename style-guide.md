# Front-end Style Guide

## Layout

The design was created to the following widths:

- Desktop: 1440px

## Colors

### Primary

- Eucalyptus: hsla(165, 59%, 50%, 1)
- Cetacean Blue: hsla(251, 76%, 10%, 1)

- Warning: rgba(244, 199, 0, 1)

- White: hsl(0, 100%, 100%)
